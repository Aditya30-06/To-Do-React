# 📝 To-Do React App

A simple and responsive To-Do application built using **React** and **Vite**.

---

## 🚀 Live Demo

👉 [Click here to view the live app](https://Aditya30-06.github.io/To-Do-React)

## 💡 Features

- Add new tasks
- Delete tasks
- Live UI updates
- Built with React 19 and Vite 7
